# Korje Hasana - যুব জামায়াত বদলগাছী

একটি ইসলামিক আর্থিক সহায়তা প্ল্যাটফর্ম যা সম্পূর্ণ সুদমুক্ত কর্জে হাসানা প্রদান করে।

## Features
- আবেদন ফর্ম (শিক্ষা, চিকিৎসা, নির্ভরশীল ফান্ড)
- দান/সাদাকা সংগ্রহ
- স্বেচ্ছাসেবক নিবন্ধন
- রিয়েল-টাইম পরিসংখ্যান
- Google Sheets/SheetDB ডাটাবেজ

## Setup

### Google Sheets Setup
1. Google Sheets তৈরি করুন
2. Apps Script ডেপ্লয় করুন
3. Script URL কনফিগার করুন

### SheetDB Setup
1. SheetDB অ্যাকাউন্ট তৈরি করুন
2. API URLs কনফিগার করুন

## Deployment
সরাসরি GitHub Pages, Netlify, বা যেকোনো static hosting-এ ডেপ্লয় করুন।
